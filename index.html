    HTML5

    Hypertext Markup Language เป็นภาษาประเภท Markup ที่ออกแบบมาเพื่อใช้ในการสร้างเว็บเพจ มีโครงสร้างการเขียนโดยอาศัยตัวกำกับ (Tag) ควบคุมการแสดงผลข้อความ รูปภาพ เสียง อื่นๆ ที่สามารถเรียกดูผ่านทางเว็บเบราว์เซอร์ได้ แต่ละ Tag สามารถระบุหรือควบคุมการแสดงผลของเว็บ ให้เป็นไปตามที่ผู้ออกแบบ

    CSS

    CSS3/SCSS

    เขียนCSS เป็นหลัก รองลงมาคือเขียน CSS Preprocessor เพราะการเขียน CSS ในรูปแบบใหม่นี้เอง ที่ทำให้เราสามารถทำอะไรได้หลายๆ อย่าง และยังทำให้เรามีระเบียบในการเขียนโค้ดขึ้นอีีกด้วย

    JQUERY

    JQUERY

    ที่เลือกใช้เพราะ เร็วกว่า javascript ธรรมดาถึง 700% ง่ายในการเรียนรู้และใช้งาน มีปลั๊กอินให้เลือกใช้มาก รองรับข้ามบราวเซอร์ได้ดี และฟังชั่นต่างๆของ Jquery ค่อนข้างครบ

    Bootstrap

    Bootstrap

    ฝั่ง Front-end Framework ผมจะใช้ Bootstrap3 ในการจัดการเว็บไซต์ เพราะจะช่วยให้การพัฒนาเว็บไซต์เร็วขึ้น ง่ายขึ้น และเป็นระบบมากขึ้น ซึ่งในที่นี้น่าจะหมายความว่า ถ้าเราใช้ Bootstrap แล้ว เราก็ไม่จำเป็นต้องไปหาอะไรมาเพิ่มอีก
